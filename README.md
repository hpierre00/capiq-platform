# CapIQ — Capital Intelligence Platform

AI-powered pre-application deal intelligence for independent real estate investors.

## What It Does
CapIQ analyzes real estate deals against real lending criteria before investors approach lenders — scoring fundability, identifying risks, and matching to the right capital source.

## Stack
- Single-file HTML frontend (dark financial intelligence aesthetic)
- Netlify serverless function (`capiq-analyze.js`) as AI proxy
- Anthropic Claude API for deal underwriting analysis

## Deploy
Requires `ANTHROPIC_API_KEY` set in Netlify environment variables.
